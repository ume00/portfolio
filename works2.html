CORDING STUDIO

  • 種別：ランディングページ

  • 制作時期：2023年
  • 制作期間：1ヶ月
  • 言語：HTML / CSS / JavaScript(jQuery) / PHP
  • 使用ツール：Photoshop / WordPress / Sublime Text

コーディング代行サービスのサイト制作を行いました。

コーディング代行サービスとしての技術力が直感的に伝わるように、視覚的に印象に残るグラデーションや動きのある表現を取り入れ、「こうした実装ができる人に依頼したい」と感じてもらえるよう、デザインとコーディングの両面から設計しました。
また、視覚的な演出だけでなく、デバイスごとの見やすさにも配慮しています。

サイトをみる

Point 1

技術的な強みを伝えるファーストビュー

ユーザーの操作に応じて反応するアニメーションや印象的なグラデーションの背景表現を取り入れ、ページを開いた瞬間から自然と関心を引くファーストビューを設計しました。
特にマウスカーソルの動きに応じて変化する幾何学模様の実装は、コーディングによる表現力や技術力を、言葉に頼らず視覚的に感じ取れるよう工夫しています。
見た目の印象だけでなく、操作したときの挙動まで含めて、技術を強みとするサービスであることが伝わる構成を目指しました。

Point 2

デバイスを問わず閲覧しやすい価格表

価格表には丸・四角・三角のオブジェクトが動くアニメーションを入れ、堅苦しくなりがちな価格表に軽やかさを加えています。
また、PCでは一覧性を重視した1つの表、スマートフォンでは縦に並べた3つの表構成に切り替えることで、横スクロールを発生させず、どのデバイスでもストレスなく価格を閲覧できる設計にしました。

Point 3

サイト全体に統一感を持たせた問い合わせ導線

サイト内の複数箇所に問い合わせへの導線を配置し、閲覧の途中でも必要なタイミングで問い合わせページへアクセスできる構成にしています。
スマートフォンでは、一定のスクロール位置以降で問い合わせボタンが画面下部に常に表示されるよう制御し、指が届きやすい位置からスムーズに問い合わせページへアクセスできるように配慮しました。

また、ボタンはファーストビューと同じ青からピンクのグラデーションを採用し、目に留まりやすさを確保しつつ、サイト全体のデザインと統一感があるようにしました。
さらに、ボタンをクリックした位置からグラデーションが広がるアニメーションを実装し、動きのある表現を取り入れながら、視覚的な楽しさも感じてもらえるよう設計しています。

Point 4

入力時の不安を減らすフォームまわりの制御

お問い合わせフォームでは、入力状態に応じたフォーカス表現や未選択項目の判別などを行い、入力状況が分かりやすくなるよう工夫しています。

必須項目のチェックやエラー時のスクロール制御も実装し、送信時にどこを修正すればよいかがすぐに分かる設計にしました。
小さなストレスを減らすことで、入力途中で迷いにくく、安心して送信まで進められる設計にしています。
