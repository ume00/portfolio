－ Scroll to View －

  • 制作時期：2025年
  • 制作期間：1ヶ月
  • 言語：HTML / CSS / JavaScript(jQuery) / PHP
  • 使用ツール：Figma / WordPress / VSCode

旅館サイトの全面リニューアルにあたり新デザインの実装依頼をお引き受けいたしました。

前任者不在で既存のWordPress環境や実装環境が不明な状態からのスタートだったため、クライアント様とやり取りを重ねながら、実装環境の確認やテスト環境の構築を行いました。
また、運営担当の方が安心して更新作業を行えるよう、WordPressの管理画面からヘッダーを含むすべての文章や画像を編集できるよう構築を行いました。
あわせて、運営マニュアルの作成、お問い合わせフォームやGTMの設置、本番環境への移行作業、基本的なセキュリティ対策など、公開後の運用を見据えた対応も行っています。

サイトをみる

Point 1

デバイスごとの閲覧環境を考慮したレスポンシブ設計

PC・タブレット・スマートフォンそれぞれの画面サイズに応じて、無理なく閲覧できるようコーディングを行いました。
サイト内の随所に登場するキャラクターについては、デザインで定められたサイズを正確に再現しつつ、画面サイズが変化しても文字やコンテンツに重ならないよう、表示位置や余白を細かく調整しています。

また、マンガでわかる...のページはスマートフォンで閲覧時のみ横スライド表示にすることで、画面サイズが限られる中でも読み進めやすい構成になるよう配慮しました。

Point 2

SEO対策・表示速度・計測環境の整備

検索エンジンから適切に認識されるよう、sitemap.xmlの作成や基本的なSEO設定を行いました。
あわせて、画像をWebP形式へ最適化し、ページ表示の負荷を抑えることで、快適に閲覧できる環境を整えています。
また、ファビコンやOGPの設定により、検索結果やSNS上でも内容が正しく伝わるよう調整しました。

さらに、GTMを導入し、予約サイトへの遷移やお問い合わせといったCTAのクリックをGA4で計測できるように設定を行い、施策の成果を数値で確認できる体制を整えました。

Point 3

安定した運営を支えるセキュリティ対策

お問い合わせフォームにはreCAPTCHAを導入し、スパム送信への対策を実施しました。
加えて、不正アクセス防止用プラグインの設定、常時SSL化、ファイアウォール設定など、複数の対策を実施しています。

一つの対策に頼らず、複数の対策を組み合わせることで、安心して運用・閲覧できる環境を構築しています。

Point 4

技術知識不要で更新できる管理画面設計

ヘッダーを含むすべての文章や画像を、WordPressの管理画面から直感的に編集できるよう構成しています。
また、「お客様の声」や「よくある質問」については、運営者様ご自身で追加・更新できるよう管理画面のカスタマイズを行いました。

制作後も情報発信を続けやすく、必要なときに運営者様ご自身で手を入れられる状態を目指した設計です。
日々の更新のたびに外部へ依頼する負担を減らし、運用面でのストレスが少なくなるよう配慮しています。
