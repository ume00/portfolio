  • 言語：HTML / CSS / JavaScript(jQuery) / PHP
  • 使用ツール：XD / Canva / VSCode / GitHub

ヨガスタジオのデモサイトを制作しました。

「開放感溢れる空間で、日常から離れた特別な時間を。」をコンセプトに、自然の中に身を置くような感覚を意識し、ブラウンやグリーンを中心としたナチュラルで落ち着いた色合いを採用したデザインを採用しています。 全体の動きはふわりとした穏やかなアニメーションで統一し、視覚的にもリラックスできる印象になるよう設計しました。

サイトをみる

Point 1

落ち着いた配色とアニメーションによる世界観づくり

ブラウンやグリーンの背景色を基調にしたナチュラルな配色と、ふわりとした柔らかいアニメーションを組み合わせることで、サイト全体に落ち着いた雰囲気を持たせています。
色使いと動きの方向性を揃えることで、ページをまたいでも印象がぶれないよう配慮し、統一感のある世界観になるよう心がけました。

Point 2

視覚的に伝えるレイアウトと導線設計

ファーストビューにはフルスクリーン動画を配置し、大自然の中で映像をイメージとして用いることで、開放感や日常から少し距離を置いた心地よい時間を連想させる導入にしました。
また、トップページ全体で画面を大胆に二分割したレイアウトを取り入れ、大きなビジュアルとシンプルな導線を組み合わせることで、情報を詰め込みすぎず、ユーザーが必要な情報へ直感的にアクセスできる構成としています。

Point 3

ヘッダーの表現と共通パーツの構造設計

ヘッダーは透過表現を取り入れ、スクロールに応じてヘッダーの文字色が切り替わる仕様にすることで、世界観を保ちながらも、コンテンツの視認性を損なわないデザインになるよう調整しました。

また、このサイトは静的HTMLで構成していますが、ヘッダーやフッターといった共通要素をパーツとして切り分けて読み込む設計としています。
これにより、共通パーツのデザインや挙動の一貫性を保ちながら、今後の調整や拡張にも対応しやすい構成となっています。

Point 4

操作感と読み込み速度の工夫

体験レッスンページには、予約チケットをモチーフにしたUIをCSSで表現し、視覚的なアクセントとして取り入れています。
また、ホバー時にはチケットの半券が切れるような動きを加えることで、自然と触れてみたくなる操作感を意識しました。

こうした動きのある表現や、インストラクター紹介ページの詳細表示切り替えなどは、できる限りCSSを中心に実装することで、動作や読み込み速度に負荷がかからないよう配慮しています。
あわせて画像はすべてWebP形式に最適化することで、全体を通して読み込み待ちによるストレスを軽減し、快適に閲覧できる構成にしました。
